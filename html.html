HTML

Hyper Text Markup Language

HTML is not a programming language. There are no conditional statements or loops etc. It's a language for making web pages and html documents. It is only presentational. Every website outputs HTML to the browser. It is used in conjunction with CSS, a styling language.

HTML tags are the basic building block of any web page.

  • Elements are surrounded by angle brackets for example: <#tagname>content<#/tagname> the tagname is the element - I'm using the # symbol in the tag or else the tag would be interpreted literally.
  • Tags usually have start and end tags, indicated by the forward slash
  • Some tags, like <#br/> close themselves (remnant of XHTML)
  1. You can use the <#strong> and <#em> tags to bold and italicize respectively, but you can change that functionality using CSS if you want.
  2. You can create links using an <#a href = ''> tag. The href part is called the attribute, and is the location where you want the link to open. It can be a local URL, like another page on your site, in which case you'd just write "about.html" as an example, or you could link to an external URL, in which case you'd write https:// etc. You can also specify that you want the page to open onto another tab, in which case you'd specify the target attribute as '_blank'. For example, this text is linked to the Google home page.
  3. All HTML tags can have attributes. Attributes provide information about an element. They are key:value pairs. href and target are two examples of attributes.
  4. Lists: The <#ul> tag stands for unordered list, while the <#ol> tag stands for ordered list.
  5. Tables: The tags for tables are shown below. Also, a table itself is shown further below.
    • <#table>
    • <#thead>
    • <#tbody>
    • <#tr> - Table Row
    • <#th> - Table Heading
    • <#td> - Table Data

  7. The <#div> tag is nothing more than a container unit that encapsulates other page elements and divides the HTML document into sections. Web developers use <#div> elements to group together HTML elements and apply CSS styles to many elements at once.
  8. <#meta> tags provide structured metadata about a Web page. They are part of a web page's head section. Multiple Meta elements with different attributes can be used on the same page. Meta elements can be used to specify page description, keywords and any other metadata not provided through the other head elements and attributes.
  9. Important <#meta> tag attributes are the title attribute, which is what your website looks like when it shows up on a search engine results page. The keywords attribute, which is basically a bunch of keywords from your website, doensn't matter as much for SEO though. The description attribute, which is what shows in normal text below your title attribute in a search engine, and is usually a concise decription of your site. The language attribute, which specifies human language (English, Spanish, etc.). The robots attribute, which specifies whether search engines and crawlers are allowed to index or scrape a page.

CSS

Cascading Style Sheets

Similar to HTML, this is not a programming language.

Three methods of adding CSS to your webpage:

  • Inline: adding it directly in the html element
  • Internal: adding it using style tags within a single document
  • External: Linking an external .css file
  • So you have a{background-color: yellow;} The three elements are called "Selector", "Property", and "Value".
  • In terms of attributes for elements, IDs should be condsidered uique. Like "main-nave bar", etc. But Classes shouldn't because you'd want to assign the css style to all elements of that class.
  • Padding is the space insed; margin is the space outside.
